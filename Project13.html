Mail Web App

Mail Web App

Mail Web Application is something like basic clone of Email giants. In mails someone can send and recieve a mail, reply on a mail, can save a mail in archieve and the basic layout makes it really easy for someone to work on it.

This project sends and recieves mails in the form of json and save data in mySQL database(currently in host computer). The frontend is develped using Html, CSS, JavaScript and Django is used as a backend server in it.
